# 의료기록 블록체인

> Version : 1.0
> Date : 2018.09.12

## 현재 문제점
- 병원진료기록이 삭제되거나 위변조될 수 있다.

## 해결 방안
- 병원진료기록과 의약품 처방 및 주문 목록을 블록체인으로   
단, 병원진료기록의 경우 개인의 프라이버시를 침해할 수 있기 때문에 복제가 불가능하고, 열람 기록이 블록에 추가되게 하며, 내용을 환자가 선택적으로 전송할 수 있게 한다.
